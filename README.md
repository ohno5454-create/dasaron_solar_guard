## v2.1 주요 변경 (v0.9 → v2.1)

1. **벡터 지도** — MapLibre GL + OpenFreeMap 무료 벡터타일(API 키 불필요). 축소 시 세계지도 전체 ~ 확대 시 읍면동·50m 축척(줌 19)까지. 오프라인/차단 시 내장 SVG 지도 자동 폴백, 카카오맵 키 입력 시 카카오맵 우선.
2. **통합관제 투명 패널** — 사이트 트리·기상정보·KPI 칩 배경을 반투명으로 변경, 뒤 지도가 비치도록.
3. **발전 모니터링 › 사이트별 출력 탭** — 보유 발전소 전체의 실시간 출력 막대그래프·표 + 관리자용 **통합 모니터링 사용자(발전주별) 집계** 창.
4. **누적 발전량 옆 누적 발전시간(h) 표시.**
5. **REC 가중치** — 계정관리 › 수익 정산 설정에서 SMP·REC·가중치 수정(전 화면 즉시 반영).
6. **사이트 상세** — 발전소 용량 표시. 인버터(INV-n)와 스트링 채널(ST-nn)을 구분, **인버터 1대당 18채널**을 각각 수동 격리/재투입, 인버터별 정지/재기동. 자동대응이 아닌 수동 원격 제어.
7. **ESS** — 충전 10:00~15:00 / 방전 15:00~익일 10:00 스케줄 자동운전, **방전 수익 창**(금일·월누적, kWh×SMP), SOH·PCS·셀온도·사이클 표시.
8. **EV 관리 O&M** — 충전/방전(V2G) 횟수·누적 용량·결제 관리(회원/비회원 단가, 금일 결제, 월 추정 매출), BMS 연동 상태(OCPP 2.0.1/ISO 15118).
9. **앱 설치 버튼 삭제** (브라우저 기본 설치 UI 사용).
10. **AI 위험도 지표 현실화** — KEC·IEC 62446-1·IEC 62109-2·UL 1699B 기반 임계값·가중치로 교체. 상세 근거·HW 수집 방안은 `docs/` 보고서 참조.
11. **A/S 페이지를 사이트 상세에 통합** (현재 발전소 기준 접수·이력).
12. **계정 등록 연락처 3개** — 메인·서브 전화번호 + 안전관리자 연락처.
13. **신규 발전소 등록** — 계정관리에서 발전소 정보(용량·유형·인버터 대수·준공일·주소·위경도·통신) 기재 + 발전주 계정 동시 발급. 지도에 즉시 반영.
14. **7개 국어** — 한국어(기본)·영어·일본어·중국어·스페인어·프랑스어·독일어.
15. **3종 배포** — 웹(PWA) / Windows(`windows/`) / Android(`android/`).

## 2차 개선 (v2.1 업데이트)

1. **카카오맵 실지도 적용** — JavaScript 키 내장, **항공(위성) 뷰 기본**. 미등록 도메인·오프라인에서는 벡터지도(MapLibre) → SVG 순으로 자동 폴백. ※ 카카오 개발자 콘솔 > 플랫폼 > Web에 배포 도메인을 등록해야 실지도가 표시됩니다.
2. **누적 발전량 표기 변경** — 총 발전시간 대신 **준공 후 1일 평균 발전시간(h/일)** 표시.
3. **가중치 잠금** — 발전 모니터링에서는 표시 전용(수정 불가), 계정관리 > 수익 정산 설정에서만 변경 가능.
4. **계정관리 + 고객관리 통합** — 하나의 "계정·고객관리" 화면으로 합침.
5. **스트링 채널 동적 생성** — 인버터별 채널 수가 고정 18이 아닌 12·18·24 등 **수신 신호에 따라 자동 생성**. 새 스트링 신호 감지 시 채널이 자동 등록되고 경보 이력에 기록.
6. **VPP 내일 발전량 예측** — 기상청 단기예보 기반(데모 시뮬레이션) 예측 창을 통합관제 우측 패널과 발전 모니터링 메인에 추가. 예상 발전량·발전시간·수익 + **VPP 입찰 권고량(P90)**.
7. **보안 강화**
   - 비밀번호 **SHA-256 해시 저장** (평문 미저장·화면 마스킹, 자동로그인도 해시 토큰만 저장)
   - 로그인 **5회 실패 시 60초 잠금**
   - **무활동 자동 로그아웃** (기본 30분, 설정에서 변경)
   - 전체 차단 등 위험 제어에 **2단계 확인**("차단" 입력)
   - **CSP**(콘텐츠 보안 정책)로 허용 출처 외 스크립트 차단
   - **보안 이벤트 로그** (로그인·제어 명령 감사 기록, 설정 화면)
   - 정식판 권장: 서버 인증(HTTPS·세션 토큰), OTP 2단계 인증, 장치 mTLS, 2인 승인 — `docs/` 보고서 참조

## 3차 개선 (v2.2)

1. **지도 레이어 정렬 수정** — 창 크기 변경·로그인·화면 전환 시 카카오맵 `relayout()`을 호출해 위성 타일과 라벨이 함께 확대·축소되도록 수정, 지도 영역도 화면에 꽉 차게 확대.
2. **발전소 등록 영속화** — 신규 등록 발전소·정보 수정이 localStorage에 저장되어 재로그인/재접속 후에도 유지.
3. **발전 모니터링** — 보유 발전소 전체 합계 아래에 발전소별 요약 칩 추가, 클릭하면 해당 사이트 상세로 이동.
4. **수익 현황 정리** — 적용 버튼·부연 설명 삭제, SMP·REC·가중치는 표시 전용(수정은 계정관리 > 수익 정산 설정).
5. **스트링 채널 차트** — 전류/전압 차트에서 인버터를 선택하면 해당 인버터의 전체 스트링 채널이 개별 시리즈로 표시.
6. **경고 이력 → 스트링 바로가기** — AI 경고 이력을 클릭하면 사이트 상세의 경보 발생 스트링 채널로 이동해 8초간 강조 표시.
7. **웹 통보(문자형 푸시)** — 자동 대응 설정·내 알림 채널에 웹 푸시(브라우저 알림) 채널 추가.
8. **경보 이력 즉시 조치** — 경보가 발생한 인버터·채널의 격리/재투입 버튼을 경보 행에 표시, 격리하면 해당 경보는 자동 '확인' 처리.
9. **보고서 경보이력 전체 표시** — 누적 이벤트·조치 내역을 제한 없이 모두 출력(인쇄 포함).
10. **RTU 저장메모리(스토어&포워드)** — 30초마다 누적 데이터를 로컬 저장하고, 통신이 끊겼다 재연결되면 끊긴 시점부터 이어서 복원(결측 구간 백필 + 동기화 알림).
